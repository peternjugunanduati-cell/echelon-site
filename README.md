Echelon Central
This is the central repository for all Echelon Interiors projects.
It serves as the master source for designs, assets, and project code.

Purpose
Centralized hub for all interior design projects.
Maintains consistent updates across multiple project repositories.
Acts as the upstream repository for project-specific forks or clones.
Usage
Clone the central repository:
git clone https://github.com/EchelonInteriors/echelon-central.git
